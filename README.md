# Jabil Wireframe 
This web-application was a basic wireframing for a potential costomer. It is a collection of views and dashboards that display visuals and quick glance information about statistics the user cares about.

## Software Lanuage 
This front-end application was written in PHP using the Laravel framework, HTML & CSS and JavaScript.
